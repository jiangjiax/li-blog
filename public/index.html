我的项目

Stars

Stars(繁星)是一个基于 Go 语言开发的去中心化个人网站生成器，支持 Web3 功能。它能帮助你快速创建一个现代化的个人网站，并提供 Web3 集成功能。本项目旨在为创作者提供一个去中心化的内容发布平台。

  • 使用 Go 和现代 Web 技术构建
  • 支持 Web3 集成，提供去中心化内容发布
  • 支持 Markdown 解析
Go Web3 网站生成器

知闻AI

知闻AI使用AI技术进行新闻内容的自动分类、评估每天超过 2000 条的最新新闻，选择最相关和最重要的信息，为每篇文章生成简明摘要，并将这些策划的文章汇编成相应的日报或周刊。

  • 使用 AI 技术进行新闻内容自动分类与摘要生成
  • 每天超过 2000 条的最新新闻
  • 提供每日、每周、每月的日报和周刊
